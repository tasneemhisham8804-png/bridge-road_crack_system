📷 Camera (USB)

Connection: USB 2.0/3.0 to Raspberry Pi

Data Rate: 30 FPS @ 1080p

  • ✓ Plug and play

  • ✓ High speed

📡 Accelerometer (I2C)

Connection: I2C Bus (GPIO 2,3)

Power: 3.3V

Address: 0x68 (7-bit)

Update Rate: Up to 8kHz

  • ✓ Low power
  • ✓ Multiple sensors possible
  • ✓ Proven protocol

💧 Moisture (SPI/ADC)

Connection: SPI via MCP3008

Power: 5V

Channel: ADC Channel 0

Resolution: 10-bit (0-1023)

  • ✓ Analog input conversion
  • ✓ 8 channels available
  • ✓ Cost effective

🌡️ Temperature (1-Wire)

Connection: 1-Wire (GPIO 4)

Power: 3.3V

Protocol: Digital sensor

Accuracy: ±0.5°C

  • ✓ Requires 4.7K pullup
  • ✓ Long cable support
  • ✓ Waterproof versions

📏 Strain (SPI/HX711)

Connection: SPI (GPIO 5,6)

Power: 5V

Resolution: 24-bit

Amplifier: HX711 module

  • ✓ High precision
  • ✓ Weight/force measurement
  • ✓ Industrial grade
